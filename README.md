# Academic_Collaboration_System
A Machine Learning-powered web application that analyzes student academic data to identify performance risk, generate personalized study recommendations, and suggest intelligent collaboration opportunities.
This project demonstrates the practical integration of AI, data engineering, and full-stack web development in an educational context.
🚀 Project Highlights
🔐 Authentication system (Login / Registration)
📊 Academic risk classification (Safe / At Risk)
🧠 Personalized AI-driven study recommendations
👥 Intelligent peer & study group suggestions
📈 Performance visualization dashboard
🌐 RESTful API architecture
🧠 Machine Learning Approach
Model: Random Forest Classifier
Objective: Predict student academic risk level
Data Pipeline
Multiple CSV datasets merged using Student ID
Data preprocessing & cleaning
Feature selection (attendance, assessments, engagement metrics)
70/30 train-test split
Model evaluation on unseen data
The model was selected for its robustness, interpretability, and ability to handle structured academic datasets effectively.
